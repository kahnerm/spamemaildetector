# spamemaildetector
Spam Email Detector, A machine learning web app that uses Multinomial Naive Bayes and NLP to classify emails as spam or not spam. Built with Python, Scikit-learn, NLTK, and Streamlit.



This is how the folder structure of this repository should be setup.

spam-detector/
├── data/
│   └── emails.csv
├── model/
│   └── (saved model files go here)
├── notebooks/
│   └── train_model.ipynb   ← Noel's workspace
├── predict.py              ← Your main deliverable
├── app.py                  ← Quojosalyn's Streamlit app
├── requirements.txt
└── README.md
